### Sensehat

```py
from sense_hat import SenseHAT

sense = SenseHAT()
sense.set_rotation(180)

message = "Raspberry Pi SIG"

yellow = [255,255,0]
red = [255, 0, 0]

speed = 0.1

while True:
    sense.show_message(text_string=message, scroll_speed=speed, text_color=yellow, back_color=red)

```
